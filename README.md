## Getting Started

### Installation

```
git clone git@github.com:SumaiaSorna/e-commerce-back-end.git
cd e-commerce-back-end
code .
npm i
```

### Launch the app

```
npm run start
```

### Tests with Coverage

```
npm run test
```

### Tests with Watch

```
npm run test:watch
```
